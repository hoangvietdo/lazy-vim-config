# lazy-vim-config
## How to install
```
rm -rf ~/.local/share/nvim
rm -rf ~/.local/state/nvim
rm -rf ~/.cache/nvim
rm -rf ~/.config/nvim
git clone https://github.com/hoangvietdo/lazy-vim-config.git ~/.config/nvim
mv -v ~/.config/nvim/lua/plugins/* ~/Videos/
nvim
mv -v ~/Videos/*.lua ~/.config/nvim/lua/plugins/
nvim
```
